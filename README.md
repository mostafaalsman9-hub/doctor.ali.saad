# منصة الدكتور علي سعد

موقع ويب Static Frontend فقط لعرض فيديوهات تعليمية، ملفات PDF، إعلانات، وجداول كورسات، مع محاكاة نظام مستخدمين ولوحة أدمن داخل نفس الصفحة.

## الملفات

```text
index.html
style.css
script.js
README.md
assets/
  images/
  icons/
```

## التشغيل

افتح ملف `index.html` مباشرة في المتصفح، أو ارفعه كما هو على GitHub Pages.

## المميزات

- تسجيل مستخدم جديد بالاسم الرباعي، الرقم القومي، رقم الهاتف، البريد الإلكتروني، وكلمة المرور.
- حالات العضوية: Pending / Approved / Blocked.
- لا يظهر المحتوى للمستخدم إلا بعد الموافقة عليه من الأدمن.
- لوحة أدمن داخل نفس الموقع بدون صفحة منفصلة.
- قبول، حظر، حذف الأعضاء.
- إضافة وحذف فيديوهات YouTube وShorts.
- إضافة وحذف ملفات PDF.
- إضافة وحذف إعلانات بصورة ونص.
- تعديل قسم عن المنصة وتواصل معنا وخلفية المنصة.
- إضافة جداول كورسات بصورة ونص ومدة من/إلى.
- إرسال إشعارات Telegram عند التسجيل وإضافة فيديو أو PDF أو إعلان أو جدول.
- تصميم عربي RTL حديث ومتجاوب للموبايل والكمبيوتر.

## التخزين

- Firebase Realtime Database لحفظ البيانات.
- LocalStorage لحفظ جلسة المستخدم الحالية.

## ملاحظات مهمة

- المشروع Frontend فقط ولا يحتوي على Backend.
- مفاتيح Firebase وTelegram موجودة داخل `script.js` حسب المطلوب، ولا تظهر داخل واجهة المستخدم.
- على GitHub Pages يعمل المشروع بدون تعديل إذا كانت إعدادات Firebase تسمح بالقراءة والكتابة من الدومين المستخدم.
