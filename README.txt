المشكلة التي تم إصلاحها:
─────────────────────────
الخطأ "Unexpected token '<'" كان يحدث لأن الكود يحاول قراءة JSON
من السيرفر لكن السيرفر كان يرجع صفحة HTML (خطأ 404) بدل JSON.

السبب: لم يكن هناك سيرفر خلفي يعالج طلبات /api/openai/...

الحل:
─────
1. تم إنشاء سيرفر Express كامل (server.js) يعالج جميع الطلبات
2. تم إضافة fetchJSON() في الكود لفحص نوع الرد قبل قراءة JSON
3. تم إضافة رسائل خطأ واضحة بدل أخطاء تقنية

══════════════════════════════════════════
  خطوات التشغيل
══════════════════════════════════════════

1. تأكد من تثبيت Node.js (v18 أو أحدث)
   https://nodejs.org

2. افتح مجلد المشروع في Terminal/CMD

3. قم بنسخ ملف البيئة:
   → انسخ ملف  .env.example  وسمّه  .env
   → افتح .env وضع مفتاح OpenAI API الخاص بك:
      OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxx

   للحصول على المفتاح: https://platform.openai.com/api-keys

4. ثبّت المكتبات المطلوبة:
   npm install

5. شغّل السيرفر:
   npm start

6. افتح المتصفح على:
   http://localhost:3000
